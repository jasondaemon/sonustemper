# Analyze

## What it does
Analyze provides inspection tools for audio (metrics, spectra, and diagnostics) and is the foundation for Noise Removal workflows.

## When to use it
- Inspect loudness and dynamics metrics.
- Check spectrum patterns before cleanup.

## Step-by-step
1) Select a track in the Library.
2) Review metrics and visualizations.
3) Use actions to open in other utilities.
