Mastering
Batch mastering with voicing, profile, tone, and export formats.
Quick start
  1. Go to Mastering and upload source files in Input.
  2. Select the files to process (Select All or checkboxes).
  3. Choose a Voicing and Strength.
  4. Pick a Loudness Profile or Manual overrides.
  5. Set Stereo Width and Guardrails if needed.
  6. Enable output formats and settings.
  7. Run Job to start the batch.
Key tips
  • Convert Only bypasses mastering stages and applies output settings only.
  • Voicing Preview renders a short clip of the first selected file.
  • Output list shows a playable preview and download links per format.
Compare
Compare source vs processed audio with waveforms and metrics.
From Mastering
  • Open a Job Output row and click Compare.
  • The Compare page loads source + selected output.
  • Use synchronized playback and zoom to compare.
Standalone
  • Open Compare and upload a source file.
  • Review metrics and waveform for the upload.
Analyze
Read-only spectrogram and diagnostic view.
From Compare
  • Switch to Analyze to inspect the same source + processed context.
  • Spectrogram and diagnostic tools will appear here.
Tagging
Edit ID3 tags and artwork for MP3s.
Working set
  • Use the browser to add mastered or imported MP3s.
  • Working set order controls track order.
Album editor
  • Edit shared album fields and per-track metadata.
  • Upload artwork once and apply to the set.
  • Save updates or download a zip of tagged files.
Voicing Profiles
Manage voicings and delivery profiles.
Create from reference
  • Upload a reference track to generate a new Voicing or Profile.
  • Use the name override to set a friendly title.
Upload JSON
  • Upload a JSON profile; it is auto-classified as Voicing or Profile.
  • Download or duplicate to edit externally and re-upload.
File Manager
Inspect and clean up sources, outputs, and profiles.
  • Browse folders by utility (Mastering, Tagging, Profiles).
  • Delete selected files or full run folders.
  • Use downloads for quick inspection.
Third-Party Tools
SonusTemper builds on these core projects.
  • FFmpeg for audio processing and analysis.
  • FastAPI for API and server routing.
  • Jinja2 for HTML templates.
  • HTMX for partial updates.
  • WaveSurfer.js for waveform rendering in Compare.
  • Mutagen for ID3 tag editing.
